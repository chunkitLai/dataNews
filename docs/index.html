“真的拜托了”

我国幅员辽阔。北起漠河南到曾母暗沙，西边是帕米尔高原，东至黑龙江、乌苏里江汇合处。随着大开发战略的步伐，西北地区光芒日盛，成为愈发难以忽视的存在。不过在小康成为常态的今天，恐怕少有人能切实体会到底层这群人生活，它绝非理想的状态。我们从西北地区省政府来信作为观察的切入口，“哀民生之多艰”——他们构成的是西北部画卷上的粗粝图腾。

惊人的信件量

西北地区囊括五个重要省份：新疆，陕西，青海，宁夏和甘肃。这些省份的政府信箱中，有殷切的求助信件，也有投诉的相关内容。据统计，在这些逐年上升的留言数量中，甘肃省留言数量最多，累计超过了一万二千条；其次是陕西省，在一万二千条左右；新疆为第三。但从人口总量上来说，五个地区中人口总量最多的是陕西，共计3864万人有余。甘肃地区的高居榜首是否说明其民生质量较低还有待商榷，但可以肯定的是，这其中的隐患是客观存在的。

不同省份不同年份的信件量，随着量的增长颜色逐渐变深，可以看出2018年始呈爆炸式增长

从日到月，从月到年……一天天过去，各省的信件量与日俱增。

每年每月每日的信件量由色块来代替，逐渐加深的颜色和越发密集的色块分布，清晰明确的表明了，从2011年的837条，到2018年达到了10124条，这些信件的背后，隐藏着多少不为我们所知的往事。

人群画像

信笺到底在说什么？来信直接与出现的各种社会问题挂钩，而这其中城建和三农问题占据了绝大多数的比例，从2011年到2018年，城建问题逐步凸显，形势变得很严峻。随着信件的积压，一段段矛盾与冲突被尘封，得不到解决，也看不到希望。

“我每年的收入也就三四万元，在市场卖菜，我们要交租金一年一万七八，每天进大门又要收一百元，一年下来又要交三万多元，我每年的收入全部交给市场还不够，若有纠纷，就有二三十个保安前来打你……省长大人，你说这叫我们老百姓怎么生活呀！”这是一个菜农的心声，从他的描述中来看，入不敷出、捉襟见肘的日子是他的常态。

无独有偶，一封来自陇南市武都区外纳镇沟渠村村民的求助性反映了相似的窘迫：他们村的住的依旧是土坯房，人均年收不过3000，被戴上“已脱贫”的帽子、连最后一点点的补贴都丧失，这彻底激怒了当地村民：“望您父母官安排专业人士来看看！”故事不同，境遇却相似。同理于务工者，最最常见的是被拖欠血汗钱，“城建”主题在所有留言中占据了25%的比例，为所有话题之首。小小的信箱寄托的是底层农民工的殷切期盼，它与省政府连接起来，小人物也有了与一省之长交流的机会。纵使这点微弱的关联难解燃眉之急，但终归是别无出路。而根据统计局最新数据，2018年我国农民工总量为28836万人，并且中西部农民工还在稳定的增加。

如果说农民工面临的是“生存”的问题，那么普通住户所要解决的则是生活中的棘手之事。纵观来信者投诉、咨询的问题，有对于低保的疑惑，有对政府征地的质询，人事变动、住房方面的比重不容小觑，外加上教育、医疗、政务、交通等主题的来信，浩如烟渺的信件让人目眩。相对于省政府各级官员，这确实是个难以消化的数量。

于是这群人开始了漫长的等待，为着一个飘渺的回音。

留言关键词

矛盾与冲突

打开留言内容，大面积的城建话题中，开发商与市民间的矛盾显得格外打眼。“公共维修基金收取五年之久为什么没有移交房管局？”“开发商交房一直拖！我该怎么办呢！”“房子漏水多次去物业办公室却无结果，说法荒唐难以接受！”这其中涉及到的不止是买房住房的问题，还有公共设施建设对民众产生的骚扰。从2011到2019年四月底的数据来看，开发商与人民生活的矛盾并未缓解，反而有愈演愈烈之势。作为卖方，开放商在双方关系中占据的主导权使买方受到权益损害的威胁，这是一大问题。

不同年份各留言主题占比变化

更何况问题远不止于城建话题。在这些话题中，有一个略显敷衍和模糊的划分类目，其热度只是略逊于“城建”。从数据能直观地看出“其他”类目下的留言量之庞大，占据了总体的17%。到底为何这个模棱两可的话题里，有如此多的留言者选择将他们的问题划分到这个部分？通过具体的抽样观察发现，“其他”话题下的留言，还是以“城建”“三农”居多，内在原因不外乎几点。首先是留言者的文化水平，这直接决定了他们是否能清晰地理解到各类目的指代含义，更何况在这种情形下，留言者是否将上书的填写任务委托给他人，也是影响填写的一个不可测因素。即便避开以上限制，一个具有一定文化水平的留言者也依旧要面临分类的难题，不管是咨询/求助/投诉涉及的是几个主题杂糅的复杂状况，还是他们一次次在所谓“相关部门”的相互推诿下陷入的两难境地，其中周折是不言而喻的。

我国人口城乡结构比较落后，乡村人口比重依然很大，“三农”相关话题在所有话题中的比重也不容忽视。打开他们的留言面板，你便不忍心作出“不食肉糜”之态：村民们冬夏天吃不上水、生产必须的化肥遭到克扣、耕地也无端荒废……许多细节体现出来“精准扶贫”的过程出现了许多差错。另外，对于农民工而言，“欠薪”是他们眼下最为紧要的问题，这是他们与包工头乃至整个建工团队的一场拉锯战。在许多维权手段都以失败告终后，他们无可奈何地将千言万语简短地浓缩在逐字逐句里，用他们略显笨拙的表达方式控诉这个社会存在的阴暗面。

这是身处在阳光下、条件优渥的幸运儿们无法体会的。

音讯，希望

理想与现实之间的距离究竟有多远？民众主要以求助和投诉为主，各占35%、34%，再其次为咨询类，占17%。

在所有的信件中，没能收到答复自然是大多数，已回复的只占不足三分之一，其中处于办理中的更加寥寥无几。这是非常无奈的常态。面对海量的信件，审查的官员有限，一省之长更是有着繁重的工作要处理……而他们能做的，只能尽量挤出一小部分精力来给出一些被选中的信件答复，这其中又得是极少数运气非常好的，才在得到回音后能顺利解决。回看那些得不到答复的求助者，只能一天天地忍受无法改变的现状，连最后一根稻草都失去的他们，该何处安放？

信件的内容与回复情况，求助和投诉占绝大部分，未回复的占绝大部分

好在回复信件在一年一年的增加。到2018年，总体回复量超过了5000封。尽管相对于收件数量，它还远远不够，但能清晰地看到行政效率的提高，也是一个值得欣慰的发现。那些收到回音的人，或许一切还是未知数，但这些珍贵的回信无疑是一针强心剂——他们是有后盾的，一切终归要得到解决。这样的结论来源于强大的制度自信。十七大以来，我们党提出了保障民生和改善民生的执政理念，从而使民生成为社会发展的热点问题。这是人民群众的迫切需求，也是善治政府的价值追求。而“民生效率”的概念应运而生，不断增多的回信是它的价值体现，也是对那些处在水深中的基层老百姓的负责。“感谢省委领导和张掖市工商管理局的各位领导。本人对处理结果非常满意，也对我们党和国家充满了信心。”这样的评价，从无到有，到越来越多，仿佛燎原的星星之火，一点点撬动国家的民生建设。这便是身居高位为官者的幸福感，它将时刻提醒人那些肩负的使命。

5000封，它还远远不够。

各省回复信件的满意度调查，百姓对收到的回音不满意居多

任重道远

事实上，“以经济建设为中心”的文化惯性和官场晋升规则，已经让中国前几十年的民生政策受到了不同程度的伤害。经济新常态的特征下，民生建设渐渐被提上台面，成为考核社会建设的重要指标。实现最大限度的生存、尊重、自由，是我国现阶段提高人民幸福指数的需求。小小的信笺寄托着留言者重若千钧的期盼，如被弃之敝屣，则是对以人为本原则的最大伤害。

死角总是存在。居庙堂之高看不到的庶民之苦、被欣欣向荣的数据掩盖的艰辛故事……把尘封的信笺揭开，触目惊心。疾苦在眼前，盛世中国绝不会坐视不管。而民生这个历久弥新的话题，旧的问题解决，新的又源源不断的冒出来……总有人为之竭尽全力、负重前行。从零到5000+，从忽视到换位思考，庞大的社会运行机制下，修复显得心有余而力不足。但其实，能做的还有很多，民生建设的能量被远远低估。

我们可以看到，城建、三农、交通、政务、教育、就业等各种问题，一直到十九大也依旧在反复强调的精准扶贫、小康建设、西部大开发……习近平总书记更是反复地在自己的讲话中提及民生建设，政府并非坐视不理。阳光背后总是有阴影的，隐患也一直存在。海量的信件中有一条留言显得格外刺眼，它是这样的：

习近平总书记多次提到的关于民生建设的关键词汇

“省长好，我是武威市凉州区一名普通农村居民，前几年乡镇干部为了完成脱贫任务，对建立养殖棚的人进行建档立卡作为贫困户，却在建成的养殖棚里什么都不养，只为了领取国家补助，甚至有些村干部年年作为低保户领取国家补助，正真的贫困户却得不到补助。而今年拆除违建，却为了完任务，将拿着国家现金盖的养殖棚硬性拆除，希望省领导能重视。”

这位留言者写下这些的语调是那么平静，仿若稀松平常。只是字里行间隐藏的，是久被世事锤炼的无奈，对本该上行下达的政策成为一桩空头张票的无奈。

在经历了多灾多难的历史后，中国人以天生含蓄收敛的性格，默默地承受生活。这种承受，也许不具有戏剧化的外壳，却依旧能在生活中焕发对客体的光芒。这便是我们的底色，这种底色，决定了我们坚韧的文化个性，即在面对一次又一次的挑战后，依然能充满希望，面对改革道路上的种种困境，内化成去改变这个世界的温柔力量。

我们能看到从零开始踽踽而行的民生建设之路，这条路，道阻且长。无数坚守在基层的人前仆后继负重前行，试图做出改变。

是的，一封封信笺上的问题是真实存在的，但我们依旧选择相信，相信这股去改变的力量，相信这个历经过千疮百孔、打破旧日枷锁、涅槃重生成就今日之繁荣昌盛的国度。

民生多艰，请记住，他们还在等待。
